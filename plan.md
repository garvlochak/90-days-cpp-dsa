# 90 Days C++ + DSA Mastery Plan

Goal: By the end of 90 days you should be able to: - Think
algorithmically - Write pseudocode first - Convert logic into C++
quickly - Solve medium level LeetCode problems comfortably

Recommended Daily Routine - 1 hr : Concept learning - 1 hr : Pseudocode
writing - 1.5 hr : Implement in C++ - 1 hr : LeetCode practice - 30 min
: Review mistakes
