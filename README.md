# 🚀 90 Days C++ + DSA Mastery Journey

This repository documents my **90-day journey of learning C++ and mastering Data Structures & Algorithms**.

The purpose of this challenge is to build **strong algorithmic thinking**, improve **problem-solving ability**, and become comfortable implementing solutions in **C++**.

The entire journey is being documented publicly to maintain **consistency, discipline, and accountability**.

---

# 🎯 Goals of This Challenge

By the end of this 90-day journey, I aim to:

• Develop strong **algorithmic thinking**
• Write **pseudocode before coding**
• Convert logic into **clean C++ implementations**
• Solve **medium-level LeetCode problems confidently**
• Build deep understanding of **core Data Structures**
• Become comfortable solving **DSA interview problems**

# 🧠 Learning Methodology

Each day follows a structured learning system:

### 1️⃣ Concept Learning

Understanding the theory behind the concept.

### 2️⃣ Pseudocode Thinking

Writing algorithm logic in pseudocode before implementation.

### 3️⃣ Implementation in C++

Converting logic into working C++ code.

### 4️⃣ Problem Solving

Solving related problems on **LeetCode**.

### 5️⃣ Mistake Analysis

Documenting mistakes to avoid repeating them.
